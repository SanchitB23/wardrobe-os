# Wardrobe OS v2.0.0 — Vision and Acquisitions Intelligence

**Release date:** 2026-07-12  
**Tag:** `v2.0.0`  
**Previous:** [v1.1.0](../../CHANGELOG.md#110--2026-07-10)

## Summary

Wardrobe OS **2.0** ships the Lifestyle Intelligence Platform core: first-class
**trips**, continuous **purchasing intelligence**, and practical **vision
workflows** — still with deterministic engines deciding and AI only explaining
([ADR-005](../adr/ADR-005-ai-does-not-decide.md)).
